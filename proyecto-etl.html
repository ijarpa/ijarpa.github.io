ETL y Pipeline de Datos en la Nube

2025 Cliente SaaS Data Engineering

El Desafío

Los datos de la empresa estaban dispersos en múltiples fuentes (CRM, ERP, redes sociales) sin un repositorio central, lo que hacía imposible tener una visión 360 del cliente.

La Solución

Se diseñó y construyó un pipeline de datos en Azure. Se usan scripts de Python en Azure Functions para extraer datos de diversas APIs (ETL). Los datos se centralizan en un Data Lake y se modelan en Azure Synapse para ser consumidos por Power BI.

  • Arquitectura de datos escalable en la nube.
  • Procesos ETL automatizados para ingesta de datos.
  • Creación de un Repositorio Único de Datos (Single Source of Truth).

Impacto y Resultados

100% Datos centralizados
360° Visión del cliente
Escalable Infraestructura a futuro
