Finanzas & Control

Dashboard de Control de Gestión & KPIs

2025 Cliente Corporativo Business Intelligence

El Desafío

La empresa dependía de múltiples archivos Excel manuales para consolidar sus estados financieros, lo que generaba un retraso de 10 días en el cierre mensual y errores recurrentes en la data.

La Solución

Implementé una arquitectura de datos moderna utilizando SQL Server como almacén central y Power BI para la visualización.

  • Extracción automática de datos (ETL) con Python.
  • Modelado de datos en estrella para optimizar DAX.
  • Diseño de dashboard interactivo con Drill-through.

Impacto y Resultados

90% Reducción tiempo de reporte
0 Errores manuales
